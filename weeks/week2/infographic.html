🚥 Matrix Multiply Unit (MXU)

TPUs nutzen ein **Systolisches Array**, um Matrix-Multiplikationen hardwareseitig zu beschleunigen.

Statt für jede Operation Daten aus Registern zu lesen/schreiben, fließen die Aktivierungen (blau) rhythmisch von links durch ein Gitter von PEs, während Gewichte (pink) von oben geladen sind.

Taktzyklus (Clock Cycle): 0 / 12

⚡ Systolischer Datenfluss (\(4 \times 4\) Grid)

Weights preloaded | Inputs streaming
Jedes PE (Processing Element) führt eine Multiply-Accumulate (MAC) Operation aus: \(C_{ij} \leftarrow C_{ij} + A_{ik} \cdot B_{kj}\)

🔑 Stateless PRNG Keys

In JAX ist die Generierung von Zufallszahlen **zustandslos**. Standard Python Random greift auf einen globalen State zu, was parallele Threads auf GPUs korrumpieren würde.

Mit `jax.random.split` klonen wir Schlüssel explizit in neue Zweige. Klicke auf die Nodes rechts, um die Struktur weiter zu spalten.
